# Cookie Shop: Simple Shopping Cart

A lightweight, single-page web application that allows users to browse a selection of delicious cookies, add them to a virtual shopping cart, and view their total order. 

This project is built using vanilla web technologies to demonstrate core front-end concepts like state management and DOM manipulation without the overhead of heavy frameworks.

---

## 💕Tech Stack

The project is intentionally kept simple and dependency-free, utilizing:

*   **HTML5** – Structured markup for the shop layout, cookie cards, and cart display.
*   **CSS3** – Clean, responsive styling to ensure the shop looks great on both desktop and mobile devices.
*   **JavaScript (ES6+)** – Client-side logic to handle adding items, updating quantities, calculating totals, and managing the cart's state.

---

## 💕 Features

*   **Product Catalog:** A clean grid layout displaying available cookies with their names, descriptions, images, and prices.
*   **Dynamic Shopping Cart:** 
    *   Add cookies to the cart with a single click.
    *   Real-time updates for item quantities and sub-totals.
    *   Automated calculation of the total price.
*   **Responsive Design:** Optimized for seamless browsing across various screen sizes.
